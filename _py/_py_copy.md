- 对于可变对象（如 `list`），浅拷贝只复制引用，不复制内容。
- 对于不可变对象（如 `int`），浅拷贝会复制值。
```python
a = {1:2, 2:[3,4,5]}  # a 是一个字典，键 2 对应的是一个列表
b = a.copy()  # 浅拷贝 a，b 现在与 a 共享列表 [3,4,5] 的引用
a[2] = 4  # 修改 a 中键 2 的值，将列表替换为整数 4

print('c = ', a)  # a = {1:2, 2:4}
print('b = ', b)  # b = {1:2, 2:[3,4,5]}

a = {1:2, 2:[3,4,5]}  # a 是一个字典，键 2 对应的是一个列表
b = a.copy()  # 浅拷贝 a，b 现在与 a 共享列表 [3,4,5] 的引用
a[2][0] = 4  # 修改 a[2]（列表）的第一个元素

print('c = ', a)  # a = {1:2, 2:[4,4,5]}
print('b = ', b)  # b = {1:2, 2:[4,4,5]}

a = {1:2, 2:3}  # a 是一个字典，所有值都是不可变类型（整数）
b = a.copy()  # 浅拷贝 a，b 现在与 a 拥有相同的键值对，但值是独立的
a[2] = 4  # 修改 a 中键 2 的值（整数 3 变成 4）

print('c = ', a)  # a = {1:2, 2:4}
print('b = ', b)  # b = {1:2, 2:3}
```
